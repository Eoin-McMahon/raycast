# raycast
Python raycasting demo using p5py

Requirements:
* Python 3 or above
* p5py

Installation:
* make sure you are using pip for python 3
* run the following command:
```bash
pip install -r requirements.txt
```

Run the code:
```bash
python3 main.py
``
